# わが家の家計簿

1日単位で支出と収入を記録し、1ヶ月の家計を管理できるシンプルなWebアプリケーションです。

## 📋 概要

「わが家の家計簿」は、ブラウザだけで動作する家計管理アプリです。サーバー不要で、データはブラウザのローカルストレージに保存されるため、プライバシーを守りながら手軽に家計を管理できます。

## ✨ 主な機能

### 📝 記録の追加・編集・削除
- **支出・収入の記録**: 日付、カテゴリ、金額、メモを入力して記録を追加
- **簡単編集**: 各記録にホバーすると編集・削除ボタンが表示されます
- **直感的なUI**: モバイルでも使いやすいレスポンシブデザイン

### 📊 月単位の管理
- **月の切り替え**: 上部の矢印ボタンで前後の月に移動
- **収支サマリー**: その月の収入・支出・残高を一目で確認
- **日別一覧**: 日付ごとに記録をグループ化して表示

### 📈 カテゴリ別の分析
- **支出内訳**: カテゴリ別の支出をバーグラフで視覚化
- **カテゴリ別集計**: どのカテゴリにいくら使ったかが一目瞭然

### 💾 データ保存
- **ローカルストレージ**: ブラウザにデータを保存（同じブラウザで開けばデータが残ります）
- **サーバー不要**: インターネット接続がなくても動作します

## 🚀 使い方

1. **ファイルを開く**: `わが家の家計簿.html` をブラウザで開きます
2. **記録を追加**: 右下の「＋」ボタンをクリック
3. **情報を入力**:
   - 支出または収入を選択
   - 日付を選択
   - カテゴリを選択
   - 金額を入力
   - メモを入力（任意）
4. **保存**: 「保存」ボタンをクリック
5. **確認**: 月の収支サマリーや日別一覧で記録を確認

### 編集・削除
- 記録にマウスをホバーすると編集（✏️）と削除（🗑）ボタンが表示されます
- 編集ボタンで記録を修正、削除ボタンで記録を削除できます
